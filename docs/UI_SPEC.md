# ngheduocsi.vn Sales App - UI/UX Spec

## Product Positioning
This is a B2B wholesale sales tool for internal sales team selling Manhae and health supplements to pharmacies, mom & baby stores, dealers, and online sellers.

This is NOT:
- A clinic app
- A doctor community
- A medical Q&A platform
- A social network

The app must feel like:
- Alibaba Seller Center
- TikTok Shop Seller Center
- KiotViet
- B2B wholesale ordering platform

## Core UX Goal
Sales users must be able to:
1. Find product fast
2. Check wholesale price and stock fast
3. Create order in under 60 seconds
4. Track order status
5. Check customer debt and purchase history

## Visual Style
- Clean B2B SaaS
- White background
- Deep navy sidebar
- Royal blue CTA
- Amber promotion highlights
- Large cards
- Clear numbers
- Mobile-first

## Colors
- Deep Navy: #0A2540
- Royal Blue: #1E40AF
- Amber: #F59E0B
- Soft Background: #F8FAFC
- Text Primary: #0F172A
- Text Secondary: #64748B
- Border: #E2E8F0
- Success: #16A34A
- Danger: #DC2626

## Typography
Use Inter or system sans-serif.
Font should feel modern, clear, and easy to read.

## Mobile UX
Mobile is the priority.
Bottom nav:
1. Trang chủ
2. Sản phẩm
3. Tạo đơn
4. Đơn hàng
5. Tài khoản

The center "Tạo đơn" button should be emphasized.

## Desktop Layout
Desktop uses:
- Left sidebar
- Top search bar
- KPI cards
- Promotion banner
- Product cards
- Recent orders
- Low stock alerts

## Main Screens
### Dashboard
Must include:
- Doanh số hôm nay
- Doanh số tháng
- Số đơn hôm nay
- Công nợ hiện tại
- Promotion banner
- Top sản phẩm bán chạy
- Đơn hàng đang xử lý
- Cảnh báo tồn kho

### Products
Each product card must show:
- Product image
- Product name
- Wholesale price
- Retail suggested price
- Estimated profit
- Stock
- Expiry date
- Add to order button

### Create Order
Flow:
1. Select customer
2. Select products
3. Change quantity
4. Apply promotion
5. Review total
6. Create order

### Orders
Show:
- Order code
- Customer
- Total
- Status
- Created date
- Sale owner

### Inventory
Show:
- Stock available
- Reserved stock
- Low stock
- Expiry date by batch

## UX Rules
- No medical/clinic visual style
- No doctor profile cards
- No Q&A community UI
- No purple social-network feeling
- Prioritize selling, ordering, promotion, stock, and speed
- Every screen should help the sales team sell faster
